NAME
====

Calculator - My first Perl6 sample distriubtion.

SYNOPSIS
========

```perl6
use Calculator;
my $c = Calculator->new( :x<10>, :y<5> );

say $c->add;
say $c->substract;
say $c->multiply;
say $c->divide;
```

DESCRIPTION
===========

Calculator is a very basic learning distribution. I found it easy to experiment with Perl6.
